# RST-I800 5G 无人机产品简介

RST-I800 无人机巡检系统允许用户利用无人机在空中实时传输视频图像，用户不仅可以通过传统的遥控器 控制无人机还可以利用网站给无人机下达简单的飞行和相机指令，用户还可以通过网站实时获取无人机的 状态信息以及位置信息。

# 产品特点

  • 内置5G CPE（数据终端），允许无人机高速传输数据
  • 搭配上30倍高清IP摄像头，无人机可以以最高4K清晰度@30fps实时传输视频
  • 无人机采用22000mAh 6S锂电池供电
